# Paradise Nursery Shopping Application

Welcome to Paradise Nursery, your one-stop shop for beautiful houseplants! This application is designed to give users a seamless plant-shopping experience.

## Features
- **Landing Page:** Beautiful introduction with a "Get Started" journey button.
- **Product Listing:** View six unique plants per category grouped into three distinct categories.
- **Redux-backed Shopping Cart:** Smoothly add, increase, decrease, or remove items from your cart dynamically.
